Unable to export CSV from analytics page

Reporter

When I click the "Export CSV" button on the Analytics page, nothing happens. The button appears clickable but no download is initiated. I've tested in both Chrome 124 and Firefox 130.

Steps to reproduce:

  1. Navigate to Analytics page
  2. Set date range to last 30 days
  3. Click "Export CSV" in the top-right actions
  4. Nothing happens — no download, no error

Expected: A CSV file should download.

Browser: Chrome 124, Firefox 130 on macOS

Hi John, thank you for reporting this. I can reproduce the issue on my end.

It looks like the export handler is failing silently when the dataset exceeds 10,000 rows. Our engineering team is working on a fix that will stream the CSV instead of building the entire file in memory.

As a workaround, you can try: narrowing the date range to 7 days — the export should work for smaller datasets.

I'll update this ticket once the fix is deployed. ETA is within 48 hours.
