BlackMoon Admin Kit is a comprehensive admin dashboard template built with modern web technologies including HTML, Angular, React, and Vue.js. It provides a complete set of UI components and page templates to help you build professional admin interfaces quickly.

BlackMoon Admin Kit comes in four framework versions: HTML/CSS (pure frontend, no build step), Angular 19, React 19 with Vite, and Vue 3 with Vite. Each version includes the same set of pages and components adapted to the framework's conventions.

Yes. Every layout and component is fully responsive. The sidebar collapses into a mobile drawer on smaller screens, and all grid layouts adapt using CSS flexbox and grid. The template also supports fluid, boxed, and full-width content layouts via the Settings panel.

Yes. Dark mode is built in using CSS custom properties. Clicking the theme toggle button in the header or user menu switches between light and dark modes. The selected mode is persisted in localStorage across page reloads.

Download the template, extract the zip, and open html/pages/dashboard.html directly in your browser — no build step needed. To preview all pages, open html/index.html. For Angular, React or Vue versions, run npm install && npm run dev to start the dev server.

No. The HTML version is a pure frontend template with no backend required. All data shown is static or mock data. You can integrate it with any REST API, GraphQL endpoint, or backend framework of your choice.

The HTML version is intentionally lean. The only third-party dependency is Chart.js v4.4 (loaded via CDN) for charts and sparklines. All other UI components — modals, drawers, date pickers, data grids, sortable tables, etc. — are built from scratch in vanilla JavaScript.

All colors are defined as CSS custom properties in assets/css/theme.css. To change the primary color, dark mode colors, or any other design token, simply edit the corresponding variable in that file. The entire UI updates automatically.

The HTML template uses its own CSS system with bm- prefixed classes. It does not depend on Bootstrap or Tailwind, so there are no conflicts. You can add either framework alongside it for specific use cases if needed.

The template is API-agnostic. Connect it to any REST or GraphQL backend by replacing mock data with fetch() calls. Angular, React and Vue versions include recommended patterns for API service layers in their documentation.
